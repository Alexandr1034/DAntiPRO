# DAntiPRO
DAntiPRO - это мой антивирус, написанный на языке Python. Для чего он нужен? DAntiPRO работает на технологии Virus Total. Данный антивирус не предотвращает заражение компьютера вирусом, он создан для других целей! Он нужен для того, чтобы проверять определённые файлы, или все диски компьютера на вирусы. Подробнее о антивирусе - читай README.
